# Color Palette

Our design system uses four distinct color palettes to convey different meanings and actions. Each palette includes carefully selected colors for backgrounds, hover states, text, and borders to ensure consistency and accessibility across all components.

## Color Palettes

## Usage

### Import

```typescript
import { type Palette } from '@/declaration/palette'
```

### Example

#### Button

```typescript
interface ButtonProps {
  palette?: Palette
  variant?: 'outline' | 'primary' | 'secondary' | 'trinary' | 'subtle' | 'plain'
}

// Example usage
<Button palette="danger" variant="primary">
  Delete Item
</Button>

<Button palette="confirm" variant="primary">
  Save Changes
</Button>

<Button palette="warning" variant="outline">
  Reset to Default
</Button>

<Button palette="solid" variant="subtle">
  Cancel
</Button>
```
